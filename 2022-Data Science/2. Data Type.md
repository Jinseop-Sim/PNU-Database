# Data Type
---
## Raw Data(원시 데이터)
> 가공되지 않은 측정 자료를 의미한다.  
> 측정 직후의 가공되지 않은 자료를 의미하며, 쓸만한 자료로 만들기 위해서는 가공을 해야한다.  

- 임의의 측정 자료이며, 구조가 없고, 에러가 있을 수 있다.
- 자동으로 Processing 할 수 있는 형태의 자료가 아니다.(Unformatted)

### Table

> _"대부분의 작업에서는 데이터를 테이블 형태로 만들기 위해  
> 데이터를 가공하고 처리하고 테이블을 만들어 분석한다."_
